Variao da quantidade de ratings

Existe diferena na variao de quantidade de ratings dos filmes lanados entre 2000-2015 e dos lanados entre 1985-2000?

Atravs de resample vamos gerar uma visualizao para analisarmos se h diferena entre a variao da quantidade de ratings nesses dois perodos.

resampleRatings = function(n) {
  x <- movies.me %>% filter(year == n)
  b = bootstrap(x$qtdRating, var, R = REP)
  CI.percentile(b, probs = c(.025, .975))
}

ratings_plot <- data.frame(year = c(), upper = c(), mean = c(), lower = c())
for (n in 1985:2015) {
  a <- resampleRatings(n)
  ratings_plot <- rbind(ratings_plot, data.frame(
    year = n,
    mean = mean(a),
    lower = a[1],
    upper = a[2]
  ))
}
ratings_plot <- ratings_plot[-1]/1e+06
# Visualizao da variao de quantidade de ratings
ratings_plot %>% ggplot(aes(x = 1:nrow(ratings_plot), y = mean)) +
  geom_point(size = 2, color = "cyan3") + 
  geom_errorbar(aes(
    ymin = lower, 
    ymax = upper), width=.3, size=1, color = "cyan3") +
  xlab("Anos") +
  ylab("Varincia da qtd. de avaliaes(milhes)") +
  scale_x_continuous(breaks = seq(2005,2015)) + 
  theme_bw() +
  theme(axis.ticks=element_blank())

Como podemos ver, a suspeita inicial se comprovou no grfico acima, a variao da quantidade de avaliaes significativamente maior para os filmes entre 80 e 00 do que para os filmes entre 01 e 15. Iremos aplicar a tcnica de bootstrap para saber exatamente quanto ser a diferena de variao. Podemos comprovar abaixo que existe uma diferena clara entre os dois perodos.

Mdia de ratings dos filmes

Qual a mdia de ratings dos filmes lanados nos ltimos 10 anos?

Para responder essa questo precisamos analisar os filmes e seus ratings dentre os anos de 2005 e 2015. importante informar que desconsideramos o ano de 2016 por ser o ano corrente dessa anlise e no termos uma amostragem significativa de dados deste ano.

resampleRatingMean = function(n) {
  x <- movies.me %>% filter(year == n)
  b = bootstrap(x$meanRatings, mean, R = REP)
  CI.percentile(b, probs = c(.025, .975))
}

ratingAll_plot <- data.frame(year = c(), upper = c(), mean = c(), lower = c())
for (n in 2005:2015) {
  a <- resampleRatingMean(n)
  ratingAll_plot <- rbind(ratingAll_plot, data.frame(
    year = n,
    mean = mean(a),
    lower = a[1],
    upper = a[2]
  ))
}

ratingAll_plot %>% ggplot(aes(x = 1:nrow(ratingAll_plot), y = mean)) +
  geom_point(size = 2, color = "cyan3") + 
  geom_errorbar(aes(
    ymin = lower, 
    ymax = upper), width=.3, size=1, color = "cyan3") +
  xlab("Anos") +
  ylab("Mdia de avaliaes") +
  scale_x_continuous(breaks = seq(2005,2015)) + 
  theme_bw() +

Como previsto inicialmente temos uma diferena muito pequena das mdias ao longo dos ltimos anos. Podemos perceber que, em geral, os filmes tem mdias de avaliaes entre 3.1 e 3.4. Isso quer dizer que em uma escala de 0 a 5, os filmes agradam seus espectadores na maioria das vezes.
